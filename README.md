# YouTube Scrapper
Capturar listagem de vídeos de determinado canal no YT

# PROJETO
O projeto será feito utilizando a plataforma GCP(Google Cloud plataform) e as seguintes API's
- API Youtube
- API Google Drive
- API Google Sheets
